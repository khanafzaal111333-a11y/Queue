Introduction
-------------
This README file describes my learning journey of Queue data structure in C++.

What is a Queue?
-----------------
A Queue is a linear data structure that follows the FIFO rule:

First In First Out

The element inserted first is removed first.

Topics I Learned
-----------------
1. Queue Basics
   - FIFO Concept
   - Front and Rear
   - Queue Operations

2. Basic Operations
   - Enqueue (Insertion)
   - Dequeue (Deletion)
   - Peek/Front
   - isEmpty()

3. Queue Implementation
   - Using Arrays
   - Using Linked List
  

4. Circular Queue
   - Wrap-around concept
   - Efficient memory usage

5. Applications of Queue
   - CPU Scheduling
   - BFS Traversal
   - Printer Queue
   - Task Scheduling

Challenges Faced
-----------------
- Understanding front and rear movement
- Handling overflow and underflow
- Circular queue logic

How I Improved
----------------
- Practiced queue operations daily
- Dry-ran queue examples
- Solved implementation problems
- Compared array and linked-list implementations

Programming Language
---------------------
C++

Conclusion
------------
Learning Queue improved my understanding of linear data structures and helped me understand real-world scheduling systems.
